mfwResponsiveWhen
directive in module mfw.responsive

Description

Directive that shows/hides DOM content depending on current matching rules or extensions.

It's implemented internally using AngularJS ngIf directive.

Usage

as attribute
<ANY mfw-responsive-when="{String}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
mfwResponsiveWhenString

Name of the rule or extension that lets the content appear on screen.
